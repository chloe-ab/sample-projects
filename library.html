How it works

This application is built with Microsoft SQL Server Express as the RDBMS, and managed with SQL Server Management Studio, representing a library database.
The database contains custom library branches, books, authors, publishers, book loans, and borrowers.

There are stored procedures designed to query:

How many copies of a given book are at a given library branch.
The names of all borrowers who do not have any books checked out.
Books due today and their borrowers for any given branch.
The total number of books loaned from any given branch.
The information of all borrowers who have more than a given number of books checked out.
The titles and number of copies of all books written by a given author and owned by a given branch.
