Blackjack Console App

A console application that acts as the dealer and allows the user to play the card game blackjack.

How it works

The game is a C# console application.

When the application is run the user is prompted for their name and how much money they've brought to the fictional casino.
If the user chooses to join a game of blackjack, they must place a bet to begin the round.
The user's monetary balance will increase, decrease, or stay the same at the end of each round depending on the outcome of the game.
At the end of each round, the user is able to play again if their balance is above zero, otherwise they must exit the game.
The user may choose to exit the blackjack game at the end of any round.
The rules of the game mimick the actual rules of blackjack, and the computer acts as the dealer.
