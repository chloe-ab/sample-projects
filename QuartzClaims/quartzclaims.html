Quartz Mining Claims Scraper

A C# application that downloads and saves Yukon public mining claims files, filters the data, and produces new tables with desired data on new and expired claims.

How it works

This program is a C# console application and utilizes three Nuget packages: NDbfReader, DocumentFormat.OpenXml, and ClosedXML.

The program downloads a zip file from the Geomatics Yukon index of quartz mining claims, which is updated frequently.
The file is saved and its contents are extracted locally.
The DataBase File (DBF) containing the claims is converted into a DataTable object.
The data is filtered for three sets of information: new claims since a given date, expired claims since a given date, and upcoming expiring claims given a future date.
All have default parameters of 30 days.
An Excel workbook is produced for each of the three sets of data, containing the relevant claims.
