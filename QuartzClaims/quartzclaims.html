Yukon Quartz Claims Scraper

A C# console application that downloads latest Yukon public mineral claims files, filters the data, and produces sorted tables of new and expired claims, providing a general overview of mineral development activity in the territory. It is deployed on Azure App Services in order to run on a weekly schedule, and a summary Excel table is automatically emailed to subscribers.

How it works

This C# console application is linked to an Azure web application using Azure App Services, and has a WebJob to run the program on a schedule.

The application downloads and locally extracts a zip file from the Geomatics Yukon index of quartz mineral claims, which is frequently updated.
The DataBase File (DBF) containing the claims information is converted into a DataTable object.
The data is filtered for three sets of information: new claims since a given past date, expired claims since a given past date, and upcoming expiring claims given a future date.
Each given date is a parameter with a default value. A data table is created for each set of filtered data.
Each filtered data table is then sorted by date first, and then by claim name. An Excel worksheet is created for each data table.
An Excel workbook containing all worksheets is sent by email to a list of subscribers on a weekly schedule.
